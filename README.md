🍔 FoodieHub – Food Ordering App

A modern and responsive Food Ordering Web Application called FoodieHub, built using React.js and Tailwind CSS. This project provides a smooth and interactive food ordering experience where users can browse food items, manage their cart, and place orders easily.

 Project Overview
- FoodieHub is designed using React.js for building dynamic and reusable components, while Tailwind CSS is used for fast, clean, and responsive UI styling.
- The application simulates a real-world food ordering system where users can explore 6 different restaurants, choose their favorite food items, add them to the cart, and complete their order through a checkout form. It also includes dark and light mode support for a better user experience.

Features

Food Ordering & Cart System

  - Browse a variety of delicious food items
  - Add items to cart with one click
  - Remove items from cart easily
  - Automatically calculate and update total price in real-time
 
Checkout System

  - Users can place their order using a simple and structured checkout form:
  Full Name,Email Address, Delivery Address, City, Phone Number
  - After filling the form, users can click “Place Order” to complete the ordering process.

Dark / Light Mode

  - Toggle using a moon/sun icon in the navbar for light/dark mode
     
Customer Reviews

  - Dedicated customer review section
  - Displays user feedback about food and service

Footer Section

  - Contact information included in footer
  - Clean and responsive layout
  - Easy access to support details
